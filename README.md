##Pytest Fundamentals Suite
This test suite demonstrates core Pytest features using simple examples. It is designed for learners to Python + Pytest.
---
##Features Covered
- **Assert Styles**: Basic assertions using Python's `assert`
- **Parametrize**: Data-driven testing with `@pytest.mark.parametrize`
- **Markers**: Custom markers like `@pytest.mark.regression`, `@pytest.mark.sanity`
- **Fixtures**: Function and session-scoped fixtures
- **conftest.py**: Shared fixtures across test files
- **Coverage**: Code coverage reporting using `pytest-cov`
- Testing Github Actions
